===========================================
PRZYKŁADOWE KOMENDY CURL DO TESTOWANIA AVATARÓW
===========================================

UWAGA: Uruchom te komendy z głównego katalogu projektu (History-API/)

ENDPOINT: POST /api/users/{userId}/avatar
- Przyjmuje RAW BINARY DATA (nie multipart/form-data)
- Content-Type: image/jpeg lub image/png
- Body: surowe dane binarne pliku

===========================================
CURL (Windows CMD)
===========================================

1. Upload avatara (JPG) dla użytkownika:
-------------------------------------------
curl -X POST "http://localhost:9080/historyapi/api/users/fe003ce8-0dae-46cb-8d01-104d1d91d4a0/avatar" ^
  -H "Content-Type: image/jpeg" ^
  --data-binary "@src/main/resources/avatars/1.jpg"

2. Upload avatara (PNG) dla użytkownika:
-------------------------------------------
curl -X POST "http://localhost:9080/historyapi/api/users/fe003ce8-0dae-46cb-8d01-104d1d91d4a0/avatar" ^
  -H "Content-Type: image/png" ^
  --data-binary "@src/main/resources/avatars/4.png"

3. Upload avatara 2.jpg:
-------------------------------------------
curl -X POST "http://localhost:9080/historyapi/api/users/fe003ce8-0dae-46cb-8d01-104d1d91d4a0/avatar" ^
  -H "Content-Type: image/jpeg" ^
  --data-binary "@src/main/resources/avatars/2.jpg"

4. Pobranie avatara użytkownika:
-------------------------------------------
curl -X GET "http://localhost:9080/historyapi/api/users/fe003ce8-0dae-46cb-8d01-104d1d91d4a0/avatar" ^
  -o downloaded_avatar.jpg

5. Usunięcie avatara użytkownika:
-------------------------------------------
curl -X DELETE "http://localhost:9080/historyapi/api/users/fe003ce8-0dae-46cb-8d01-104d1d91d4a0/avatar"

===========================================
POWERSHELL
===========================================

1. Upload avatara (JPG):
-------------------------------------------
$headers = @{
    "Content-Type" = "image/jpeg"
}
$body = [System.IO.File]::ReadAllBytes("src/main/resources/avatars/1.jpg")
Invoke-RestMethod -Uri "http://localhost:9080/historyapi/api/users/fe003ce8-0dae-46cb-8d01-104d1d91d4a0/avatar" -Method Post -Headers $headers -Body $body

2. Upload avatara (PNG):
-------------------------------------------
$headers = @{
    "Content-Type" = "image/png"
}
$body = [System.IO.File]::ReadAllBytes("src/main/resources/avatars/4.png")
Invoke-RestMethod -Uri "http://localhost:9080/historyapi/api/users/fe003ce8-0dae-46cb-8d01-104d1d91d4a0/avatar" -Method Post -Headers $headers -Body $body

3. Pobranie avatara:
-------------------------------------------
Invoke-RestMethod -Uri "http://localhost:9080/historyapi/api/users/fe003ce8-0dae-46cb-8d01-104d1d91d4a0/avatar" -Method Get -OutFile "downloaded_avatar.jpg"

4. Usunięcie avatara:
-------------------------------------------
Invoke-RestMethod -Uri "http://localhost:9080/historyapi/api/users/fe003ce8-0dae-46cb-8d01-104d1d91d4a0/avatar" -Method Delete

===========================================
NOTATKI
===========================================
- Zastąp UUID użytkownika (fe003ce8-0dae-46cb-8d01-104d1d91d4a0) rzeczywistym UUID użytkownika z bazy
- Avatary są zapisywane w katalogu zdefiniowanym w web.xml (avatar.storage.dir)
- Domyślnie: C:/temp/avatars
- Format pliku: {userId}.{extension} (np. fe003ce8-0dae-46cb-8d01-104d1d91d4a0.jpg)
- Endpoint przyjmuje RAW BINARY DATA, nie multipart/form-data
- Content-Type określa typ pliku: image/jpeg lub image/png
